Navigation shortcuts

  • incident.list opens an Incident list.
  • incident.do opens a new Incident form.
  • sys_db_object.list opens table definitions.
  • sys_dictionary.list opens field metadata.
  • nav_to.do?uri=... deep-links to a target page.
  • stats.do, cache.do, and diagnostics pages are advanced admin tools; use carefully.

Troubleshooting path

  1. Confirm user/persona and environment.
  2. Identify table and record.
  3. Check ACLs and roles.
  4. Check UI Policy/Client Script/Form/View.
  5. Check Business Rules/Flows/Events.
  6. Check logs, email, ECC Queue, or integration records.
