# NowSkill ServiceNow Bible Portal

A free, static ServiceNow learning website from administrator to developer to architect.

## Structure

- `index.html` — homepage
- `modules/` — major learning destinations
- `volumes/volume-0/` — publication-style chapter pages
- `assets/css/nowskill.css` — visual system
- `assets/js/app.js` — command palette, expand/collapse, copy-code helpers
- `references/` — source manuscript/reference material
- `docs/` — GitHub deployment and contribution guidance

## Local preview

Open `index.html` directly, or run:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
